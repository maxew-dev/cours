Programme prévisionnel :

Durée du cours : 21H (3 jours)
  • Apports théroriques : Généralités, HTML, CSS
  • Exercices
  • Mise en pratique : Réalisation du sujet, intégration
  • Présentation et rendu des travaux, Évaluation

Objectifs pédagogiques :

  • Mettre en place un environnement de travail HTML/CSS "from scratch"
  • Concevoir et réaliser un site web statique
  • Préparer les éléments (fichiers, médias...) et les organiser de façon pertinente
  • Écrire du code HTML et CSS (respect des bonnes pratiques)
  • Rechercher les références appropriées pour compléter mes connaissances

Sujet - Tronc commun (3 jours)

Réaliser un site qui présente un processus pouvant être décrit en différentes étapes (ex : recette, tuto, mode d'emploi...).
  • Imaginer et maquetter la navigation entre les pages, ainsi que les fonctionnalités qui pourraient s'avérer pertinentes en réponse au sujet
  • Le livrable doit être optimisé, 'zippé', puis transmis par les outils collaboratifs
  • Qualités attendues pendant le cours : Créativité, Rigueur/Organisation, Attitude proactive

Programme prévisionnel :

Durée du cours : 35H (5 jours)
  • Apports théroriques avancés : HTML, CSS
  • Exercices
  • Mise en pratique : Réalisation du sujet, intégration
  • Présentation et rendu des travaux, Évaluation

Objectifs pédagogiques :

  • Confirmer les bases acquises précédemment
  • Approfondir ses connaissances en HTML et CSS
  • Utiliser les outils de déboggage

Sujet - Développeurs

Réaliser un site "catalogue" (ex : e-commerce, collection, portfoliot...).
  • Le site devra être responsive, orienté "mobile-firts"
  • Il devra comporter minimum 4 pages :
    • page d'accueil / présentation
    • galerie de visuels
    • page(s) de détail
    • un formulaire
  • Imaginer et maquetter la navigation entre les pages, ainsi que les fonctionnalités qui pourraient s'avérer pertinentes en réponse au sujet
  • Le livrable doit être optimisé, 'zippé', puis transmis par les outils collaboratifs
  • Qualités attendues pendant le cours : Créativité, Rigueur/Organisation, Attitude proactive
